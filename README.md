# DIAMOND-LANG
A custom scripting language built from scratch.

Overview:

Diamond Lang is a dynamically typed toy language made for fun. It's simple, desi-style, and expressive.

🪩 Program Structure
kaam dost hello()
    bata dost "Hello World"
bas dost
🔤 Variables
Use bana dost to declare variables. Reassign like normal.

bana dost a = 10
bana dost b = "Diamond"
a = a + 5
🔢 Data Types
Numbers: 10, 3.14
Strings: "hello", 'ok'
Booleans: sahi, galat
Null: khaali
bana dost x = 25
bana dost msg = "yo"
bana dost flag = sahi
bana dost empty = khaali
🔁 Loops
Use jab tak dost (condition) for while loops.

bana dost i = 0
jab tak dost (i < 5)
    bata dost i
    i = i + 1
bas dost
🔀 Conditionals
agar dost (cond)
nahi to dost (cond)
bana dost x = 10
agar dost x < 5
    bata dost "small"
nahi to dost 
      bata dost "large"
bas dost
📦 Functions
kaam dost square(x)
    wapas dost x * x
bas dost

bana dost ans = bula dost square(5)
bata dost ans
🔊 Output
bata dost "Hello"
likh dost "World"
bata dost 5 + 5
